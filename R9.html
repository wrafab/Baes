Rotation 9: Raspberry Pi



I quite liked my Raspberry Pi experience. I see so much potential from it. It is small, versatile, and cheap.
I learned a bit about physical computing. You have to manipulate both software and hardware in order to do that.
Raspberry Pi is nothing but a small silicon lump without OS or programs. Scratch and Python can actually tell it
what to do and how to do. I haven't done any Raspberry Pi project yet, but I feel like I want to do the LED fan clock.
